# ターゲット像

## 1. デモグラフィック

### 年齢層

- 不動産売却経験者: **60代が最多で30.9%**、次いで50代、40代
- 空き家所有者: **50歳以上が94.3%**を占める極めて高齢な構成
- 75歳以上の所有者では約33%が「20年以上」放置状態

### 男女比

- **男性約52%、女性約48%** でほぼ均等

### 居住エリア

- 空き家の**51.5%が大都市圏外**（地方）
- 89.7%が一戸建て物件
- 地方ほど築古・管理不全の傾向が強い

## 2. 売却のトリガー

### 売却理由ランキング

| 順位 | 理由 | 割合 |
|------|------|------|
| 1位 | 住み替え | 35.6% |
| 2位 | 資産整理 | 23.0% |
| 3位 | 相続 | 13.9% |
| 4位 | 不要物件の処分 | 19.85% |
| 5位 | 離婚 | 10.96% |

### 各イベント詳細

**相続（13〜15%）**
- 空き家全体の**約58%が相続により取得**
- 築55年以上の物件では**70.8%が相続取得**
- 相続前に対策を講じていた所有者はわずか23%

**離婚（約10〜11%）**
- 財産分与の必要性から売却を急ぐ傾向

**住み替え（35〜40%）**
- 最大ボリューム。家族構成変化、転勤等

**空き家の管理負担**
- 所有者の**76.9%が弊害を認識**しつつ**43.8%が何もしていない**

## 3. 難あり物件オーナーの心理・行動

### 主な悩み

1. 「不用品の整理・撤去ができていない」
2. 「何をするのがよいかわからない」
3. 「活用する具体的なアイデアがない」

### 放置の理由

1. **負動産問題（43.5%）**: 「買い手/借り手が見つからない」
2. **コスト問題（30.5%）**: 「活用・処分にお金がかかる」

### 情報収集の方法

- **インターネット検索が主流**（35.7%）
- 「知人や友人」17.8%
- 40代以上は「Google検索」「不動産会社サイト」「一括査定サイト」
- 20～30代は「YouTube」「X」「Instagram」
- 専門サービスの利用は少数派（不動産業者に管理委託は1%未満）

### 行動の障壁

1. **「何から始めればいいかわからない」** — 最大の壁
2. コスト不安（リフォーム・解体・残置物処分）
3. 心理的抵抗（訳あり物件の告知義務）
4. 共有者間の合意形成
5. 時間と手間（遠方の空き家）

### 「諦めている」人の割合

- **「どんな条件でも売却しない」: 31.3%**
- **「何もしていない」: 43.8%**
- 60代が最も行動していない（61.2%）
- ただし**「初期費用がかからない」なら売却検討する人は24.0%**
- **「手続きに時間と手間がかからない」なら20.5%**
- → **条件次第で動く潜在層が確実に存在**

## 4. Meta広告ターゲティング設計

### 基本設定

| 項目 | 推奨 |
|------|------|
| 年齢 | コア: 50〜70代 / サブ: 40〜50代 |
| 性別 | 男女均等 |
| 地域 | 地方都市重点 + 首都圏・関西圏の相続発生エリア |

### 興味関心カテゴリ

**直接的:**
- 不動産、不動産売却、不動産査定、マンション、一戸建て、土地

**相続・法律:**
- 相続、遺産相続、相続税、弁護士、司法書士、終活

**資産・財務:**
- 資産運用、老後資金、固定資産税

### ライフイベント

| ライフイベント | 活用方法 |
|---------------|----------|
| 最近引っ越した | 旧居売却ニーズ |
| 転職 | 転居に伴う売却 |
| 子供がいる（年代別） | 家族構成変化 |

### 推奨広告セット構成

**セットA: 相続・空き家層**
- 50〜70歳 / 相続 OR 終活 OR 固定資産税

**セットB: 訳あり物件・諦め層**
- 45〜70歳 / 不動産売却 OR 不動産査定 OR リフォーム / 地方都市重点

**セットC: ライフイベント起因層**
- 35〜55歳 / 最近引っ越した OR 転職 / 不動産関連興味

**セットD: ブロード配信（ML最適化）**
- 40〜70歳 / ターゲティングなし / CVデータ蓄積後はこちらが最も効率的になる可能性
