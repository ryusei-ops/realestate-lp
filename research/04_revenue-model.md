# 報酬モデル・単価感

## 1. 一括査定サイトのビジネスモデル

### 課金方式: 反響課金制（成果報酬型）

- 初期費用・月額固定費・掲載費はゼロ
- 査定依頼（反響）が1件届くたびに課金

### リード単価の相場

| 項目 | 金額 |
|---|---|
| 反響単価の相場 | **10,000円〜15,000円/件** |
| 業界平均 | **約13,000円/件** |
| イエウール | 約15,000円/件 |
| HowMa | 10,000円〜/件 |

1件の査定依頼は最大6社に同時送客 → **サイト運営側は1リードから最大6万〜9万円の売上**

### 媒介契約獲得までの実質コスト（不動産会社視点）

- 反響から媒介契約への転換率: **2%〜10%**
- 媒介契約1件の獲得CPA: **20万〜65万円**

## 2. 仲介手数料の構造

### 宅建業法の上限（税込）

| 売買価格 | 料率 |
|---|---|
| 200万円以下 | 5.5% |
| 200万〜400万円 | 4.4% |
| 400万円超 | 3.3% |

速算式（400万円超）: 売買価格 x 3% + 6万円 + 消費税

### 2024年7月施行の特例

売買価格800万円以下 → 売主・買主それぞれから**最大33万円（税込）**まで受領可能

## 3. リード獲得ビジネスの報酬パターン

### A. リード販売型（送客1件○円）

- **単価: 10,000〜20,000円/件**
- 一括査定サイトの主流モデル
- メリット: 安定した売上、成約有無に関係なく課金
- デメリット: リード品質が低いと不動産会社が離反

### B. 成果報酬型（成約時に○%）

- **相場: 手数料の10〜30%、または成約価格の0.5〜1%**
- メリット: 不動産会社のリスクが低い
- デメリット: 回収まで3〜6ヶ月以上、トラッキングが困難

### C. 月額固定型

- **相場: 月額3万〜10万円**
- メリット: 予測可能な固定収入
- デメリット: 反響ゼロでも課金、解約リスク高い

### D. ハイブリッド型

- 月額固定（低額）+ 反響課金の組み合わせ

### 広告CPA相場（参考）

| 成果地点 | CPA相場 |
|---|---|
| 売買仲介・物件問い合わせ | 40,000〜55,000円 |
| 売買仲介・会員登録 | 15,000〜25,000円 |
| 不動産投資・資料請求 | 30,000〜50,000円 |

## 4. 難あり物件の売却価格帯

### 再建築不可物件の相場

| 条件 | 通常価格比 |
|---|---|
| 43条2項で再建築可能（誓約書不要） | 約9割 |
| 43条2項で再建築可能（誓約書必要） | 約7割 |
| 隣地借用で再建築可能性あり | 約6割 |
| 再建築困難（接道なし等） | 約5割 |

買取業者に売却する場合: 市場価値の**70〜80%**。難あり物件はさらに低く**50〜70%**。

### 1件あたりの仲介手数料目安

## 5. 本プロジェクトへの示唆

最も現実的な初期モデルは**リード販売型（1件10,000〜15,000円）**。

難あり物件に特化する場合、不動産会社にとって1件の仲介手数料が23万〜56万円程度。リード単価10,000〜20,000円は十分にペイする。

成約報酬型は管理コストが高いため、まずは反響課金型でスタートが無難。
